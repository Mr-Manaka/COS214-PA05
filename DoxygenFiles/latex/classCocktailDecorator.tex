\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classCocktailDecorator}{Cocktail\+Decorator}} class is an abstract class that serves as a base for all cocktail decorators. 

It inherits from the \mbox{\hyperlink{classCocktail}{Cocktail}} class and provides a common interface for all decorators to implement. The class contains a pointer to a \mbox{\hyperlink{classCocktail}{Cocktail}} object that it decorates.
