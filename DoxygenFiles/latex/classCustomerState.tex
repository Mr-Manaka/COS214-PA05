\doxysubsection{Detailed Description}
The base class for all customer states. 

This class defines the interface for all customer states. It contains pure virtual functions that must be implemented by derived classes. It also has a protected member variable \textquotesingle{}state\textquotesingle{} which stores the current state of the customer.
