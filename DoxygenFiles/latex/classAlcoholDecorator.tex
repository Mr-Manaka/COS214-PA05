\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classAlcoholDecorator}{Alcohol\+Decorator}} class is a concrete decorator that adds an alcohol to a cocktail. 

It inherits from the \mbox{\hyperlink{classCocktailDecorator}{Cocktail\+Decorator}} class and overrides its \mbox{\hyperlink{classAlcoholDecorator_a9503a2290eb26207dc429bfb55df510b}{get\+Description()}} method. It also has a protected member variable that holds the description of the added alcohol.
