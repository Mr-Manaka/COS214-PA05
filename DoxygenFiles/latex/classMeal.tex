\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classMeal}{Meal}} class is an abstract base class that defines the interface for creating meals. 

This class contains pure virtual functions for creating meals and getting the name of the meal. It also contains a protected member variables for the name of the meal, whether it has a garnish, and whether it has a sauce. The class also defines an equality operator for comparing meals.
