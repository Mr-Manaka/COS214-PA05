\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classSplit}{Split}} class is a concrete implementation of the \mbox{\hyperlink{classTableState}{Table\+State}} interface. 

This state represents a table that has been split into two separate tables. It allows for the table to be occupied, freed, combined with another table, or split into two tables again.
