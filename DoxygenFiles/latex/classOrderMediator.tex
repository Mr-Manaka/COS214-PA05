\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classOrderMediator}{Order\+Mediator}} class is an abstract class that defines the interface for a mediator between the \mbox{\hyperlink{classWaiter}{Waiter}}, \mbox{\hyperlink{classChef}{Chef}}, and \mbox{\hyperlink{classInventory}{Inventory}} classes. 

It provides methods for notifying the mediator when an order is placed, when a plate is ready, and when drinks are ready. It also provides methods for adding a waiter and setting the inventory.

\begin{DoxyNote}{Note}
This class is meant to be inherited from and its methods implemented. 
\end{DoxyNote}
