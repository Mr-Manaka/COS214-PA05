Detailed Description

The Chef class is an abstract base class that defines the interface for creating meals and receiving orders.

It contains a protected member variable for the chef's name and a static member variable for the plate. The class provides pure virtual functions for creating starters and mains, adding meals, and receiving orders. It also provides a static function to set the plate.
